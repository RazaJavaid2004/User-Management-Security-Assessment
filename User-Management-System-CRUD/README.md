# User Management using Node, Express, and MongoDB

This is a simple user management API that allows you to perform CRUD

# Features
- Create, read, update, and delete user data

# Installation
To run this on your local machine, you will need to have Node.js and MongoDB installed. Once you have these installed, follow these steps:

1. Clone this repository
2. Run  `npm install` to install dependencies
3. Create a `config.env` file with your MongoDB connection string
4. Run `npm start` to start
